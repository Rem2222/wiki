# Find 100% Crypto Pumps 🔍 — Moonin Papa

Видео Аарона Дишнера (Moonin Papa) о бесплатном крипто-сканере на [bettertrader.io](https://bettertrader.io) и стратегии торговли послепамповой волатильности.

## Основная идея

**Не искать пре-памп** (монеты ДО взлёта) — это ineffective. Вместо этого:
1. Находить монеты/токены, которые **уже пам панули на 100%+** через сканер
2. Торговать **follow-up volume** — волатильность после пампа даёт лучшие точки входа
3. Использовать sorting по **spread** (high-to-low) + **near 24h low** для re-entry

## Как работает сканер

1. Выбрать биржу с низким объёмом (OKX, Gate.io — не Binance/Coinbase)
2. Сортировать по **spread** (размах low-high за 24ч) — high to low
3. Доп. сортировка по **near 24h low** — монеты около нижней границы после пампа

## Демо: IGU/USDT
- Памп +163%, затем вторая волна
- Ликвидность ~$1,500 USDT → макс $150 на сделку (10%)
- 15-min / 30-min таймфреймы для свингов
- Использовать TBO индикатор + 3Commas для split targets

## Выводы
- Сканер бесплатный, без регистрации, без трекинга
- Ценность: идентификация волатильных монет после взлёта
- Не предсказание, а реакция на уже произошедшее движение
