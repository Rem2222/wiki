# qmd

Локальная поисковая система для markdown-файлов. Гибридный BM25/векторный поиск с реранжированием с помощью LLM.

## Возможности

- **BM25 + векторный поиск** — гибридный подход
- **Реранжирование LLM** — переупорядочивание результатов
- **CLI** — LLM может вызывать через терминал
- **MCP-сервер** — LLM может использовать как нативный инструмент
- **Работает локально** — без облака

## Когда нужно

На малых масштабах (~100 страниц) достаточно index.md. Но по мере роста wiki нужен полноценный поиск.
