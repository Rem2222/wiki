# MCP для 1С — Настройка

> Stub-страница. Заполнить при наличии инструкции по настройке MCP-сервера для 1С.

## Что это

MCP-сервер для 1С предоставляет семантические инструменты для работы с кодом 1С:
- `get_module_structure` — структура модуля
- `get_method_call_hierarchy` — иерархия вызовов
- `get_metadata_details` — метаданные объектов
- `validate_query` — валидация запросов 1С
- `bsl_scope_members` — автодополнение

## Зачем

Согласно [[mcp-vs-builtin-token-analysis|Analysis от Антона Минякова]], MCP экономит **77% токенов** на семантических задачах в 1С.
