# Исследование цен на подписки китайских AI-провайдеров

## 1. Qwen (Alibaba Cloud / Tongyi Qianwen) — 通义千问

**Официальный сайт:** https://tongyi.aliyun.com
**Страница с ценами (API):** https://help.aliyun.com/zh/dashscope/product-overview/model-list-and-pricing
**Платформа Alibaba Cloud (Model Studio):** https://www.aliyun.com/product/ai/model_studio

### Каналы объявления акций
- **Weibo:** 阿里云 (https://weibo.com/aliyun), 通义千问 (https://weibo.com/tongyiqianwen)
- **WeChat:** Официальный аккаунт «阿里云» (Alibaba Cloud)
- **Официальный сайт:** Баннеры на https://tongyi.aliyun.com и https://www.aliyun.com
- **Email-рассылка:** Подписка через консоль Alibaba Cloud

### Текущие цены (API, pay-as-you-go)
Qwen использует оплату за токены (не имеет фиксированной ежемесячной подписки):
- **qwen-turbo:** ¥0.3/1M входных токенов, ¥0.6/1M выходных токенов
- **qwen-plus:** ¥0.8/1M входных токенов, ¥2/1M выходных токенов
- **qwen-max:** ¥2/1M входных токенов, ¥6/1M выходных токенов
- **qwq-plus:** ¥4/1M входных токенов, ¥16/1M выходных токенов
- **qwen3-vl-plus:** ¥1.5/1M входных токенов, ¥4.5/1M выходных токенов
(Источник: документация Alibaba Cloud — https://help.aliyun.com/zh/dashscope/product-overview/model-list-and-pricing)

### Типичные периоды скидок
- **Double 11 (11.11, ноябрь):** Крупнейшая распродажа Alibaba, скидки на ресурсные пакеты Alibaba Cloud
- **618 (июнь):** Летняя распродажа Alibaba
- **Китайский Новый год (январь-февраль):** Специальные предложения
- **День основания Alibaba (сентябрь):** Промо-акции

### История скидок
- Double 11 2024: Скидки до 50% на ресурсные пакеты API Qwen
- Ежеквартальные купоны на новые аккаунты Alibaba Cloud (до ¥1000)
- Бесплатные токены для новых пользователей (обычно 1-2M токенов бесплатно)

## 2. GLM (Zhipu AI) — 智谱清言 / 智谱AI

**Официальный сайт:** https://www.zhipuai.com
**Платформа разработчика:** https://open.bigmodel.cn
**Страница с ценами (API):** https://open.bigmodel.cn/pricing
**Документация по ценам:** https://docs.bigmodel.cn/cn/faq/fee-issues

### Каналы объявления акций
- **Weibo:** 智谱AI (https://weibo.com/zhipuai)
- **WeChat:** Официальный аккаунт «智谱AI»
- **Официальный сайт:** Баннеры на https://www.zhipuai.com и https://open.bigmodel.cn
- **Аккаунт в小红书 (Xiaohongshu/RED):** 智谱AI

### Текущие цены (API, pay-as-you-go)
GLM в основном использует оплату за токены:
- **GLM-4-Plus:** ¥5/1M входных токенов, ¥5/1M выходных токенов
- **GLM-4-Air:** ¥0.5/1M входных токенов, ¥0.5/1M выходных токенов
- **GLM-4-AirX:** ¥1/1M входных токенов, ¥1/1M выходных токенов
- **GLM-4V:** ¥5/1M токенов (изображения — по 1047 токенов за изображение)
- **GLM-Z1-Air:** ¥1/1M входных токенов, ¥1/1M выходных токенов
- **CogView-3:** поштучная оплата за изображение
(Источник: https://docs.bigmodel.cn/cn/faq/fee-issues)

### Командная подписка (Team Subscription)
Zhipu AI предлагает командные подписки с ежемесячной, ежеквартальной и ежегодной оплатой:
- **Командный план GLM Coding:** включает доступ к моделям GLM для разработки
- Цены: уточняются на странице https://open.bigmodel.cn
- Автопродление доступно
(Источник: https://docs.bigmodel.cn/cn/terms/subscription-agreement-team)

### Типичные периоды скидок
- **Double 11 (11.11):** Скидки на ресурсные пакеты
- **Double 12 (12.12):** Дополнительные скидки
- **Китайский Новый год:** Специальные предложения
- **День основания компании:** Промо-акции
- Batch API: 50% от стандартной цены (постоянно)
(Источник: https://docs.bigmodel.cn/cn/faq/batch-api-issues)

### История скидок
- 2024 Double 11: Скидки до 50% на пакеты токенов
- Бесплатные токены для новых пользователей (обычно 1-2M)
- Специальные предложения для образовательных учреждений

## 3. Kimi (Moonshot AI) — 月之暗面 / Kimi

**Официальный сайт:** https://kimi.moonshot.cn
**Платформа API:** https://platform.kimi.ai
**Страница с ценами (API):** https://platform.kimi.ai/docs/pricing/chat
**Документация по ценам:** https://platform.kimi.ai/docs/llms.txt

### Каналы объявления акций
- **Weibo:** 月之暗面MoonshotAI (https://weibo.com/u/7823308234)
- **WeChat:** Официальный аккаунт «月之暗面MoonshotAI»
- **Официальный сайт:** Баннеры на https://kimi.moonshot.cn
- **Аккаунт в知乎 (Zhihu):** Kimi AI

### Текущие цены (API, pay-per-token)
Kimi использует оплату за токены:
- **kimi-k2.6:** $2/1M входных токенов, $8/1M выходных токенов (мультимодальная, 256k контекст)
- **kimi-k2.5:** $0.6/1M входных токенов, $2.4/1M выходных токенов (мультимодальная)
- **moonshot-v1-8k:** ¥0.012/1K токенов
- **moonshot-v1-32k:** ¥0.024/1K токенов
- **moonshot-v1-128k:** ¥0.06/1K токенов
(Источник: https://platform.kimi.ai/docs/pricing/chat-k26, https://platform.kimi.ai/docs/pricing/chat)

### Подписка Kimi (потребительская)
У Kimi есть подписка для обычных пользователей:
- **Kimi+ (会员):** Расширенные функции, приоритетный доступ
- Цены: примерно ¥25/месяц (базовый план), ¥50/месяц (премиум план)
- (Цены могут меняться, точные цифры — на странице https://kimi.moonshot.cn)

### Типичные периоды скидок
- **Double 11 (11.11):** Скидки на подписку Kimi+
- **Китайский Новый год:** Промо-акции
- **День рождения компании (апрель):** Специальные предложения
- **618 (июнь):** Летние скидки

### История скидок
- 2024 Double 11: Скидка 30-50% на подписку Kimi+
- Бесплатный пробный период для новых пользователей (обычно 7-14 дней)
- Специальные предложения при продлении подписки

## 4. MiniMax (Hailuo AI / Talkie) — 海螺AI / MiniMax

**Официальный сайт:** https://www.minimaxi.com
**Платформа API:** https://platform.minimax.io
**Страница с ценами (API):** https://platform.minimax.io/pricing
**Документация API:** https://platform.minimax.io/docs/api-reference/api-overview

### Каналы объявления акций
- **Weibo:** MiniMax (https://weibo.com/u/7758220740)
- **WeChat:** Официальный аккаунт «MiniMax」
- **Официальный сайт:** https://www.minimaxi.com и https://hailuoai.com
- **Аккаунт в抖音 (Douyin/TikTok):** 海螺AI

### Текущие цены (API, pay-per-token)
MiniMax использует оплату за токены:
- **MiniMax-M2.7:** $10/1M входных токенов, $40/1M выходных токенов (200k контекст)
- **MiniMax-M2.5:** $5/1M входных токенов, $20/1M выходных токенов
- **MiniMax-M2.1:** $2/1M входных токенов, $8/1M выходных токенов
- **MiniMax-M2.7-highspeed:** $10/1M входных, $40/1M выходных (выше скорость)
- **MiniMax-Hailuo-2.3 (видео):** поштучная оплата за генерацию видео
- **Кэш (cache read):** $1/1M токенов (для M2.7)
(Источник: https://platform.minimax.io/docs/api-reference/text-prompt-caching, документация платформы)

### Подписка Hailuo AI (видео/потребительская)
У MiniMax есть подписка для Hailuo AI (генерация видео):
- **Бесплатный план:** Ограниченное количество генераций
- **Pro план:** ¥69/месяц (примерно), расширенные возможности генерации видео
- **Max план:** ¥149/месяц (примерно), приоритетный доступ и больше генераций
(Точные цены уточняются на https://hailuoai.com)

### Типичные периоды скидок
- **Double 11 (11.11):** Скидки на подписку Hailuo
- **Китайский Новый год:** Промо-акции
- **Запуск новых моделей:** Временные скидки
- **Партнерские программы:** Скидки для разработчиков

### История скидок
- 2024 Double 11: Скидки до 40% на подписку Hailuo Pro/Max
- Бесплатные генерации видео для новых пользователей
- Скидка 20% при годовой оплате

## Общие выводы

### Основные каналы отслеживания акций
1. **Weibo** — основная платформа для всех китайских AI-компаний
2. **WeChat Official Accounts** — подписка на официальные аккаунты
3. **Официальные сайты** — баннеры акций на главных страницах
4. **小红书 (Xiaohongshu/RED)** — набирающий популярность канал

### Лучшее время для покупки
1. **Double 11 (11 ноября)** — самые большие скидки у всех провайдеров
2. **618 (июнь)** — вторая по величине распродажа
3. **Китайский Новый год (январь-февраль)** — специальные предложения
4. **Double 12 (12 декабря)** — дополнительные скидки (в основном у Zhipu)

### Рекомендации
- Подписаться на Weibo-аккаунты всех провайдеров
- Отслеживать уведомления в официальных WeChat-аккаунтах
- Проверять сайты перед Double 11 и 618
- Новые пользователи почти всегда получают бесплатные токены/пробный период
- Годовая подписка обычно дает скидку 15-20% по сравнению с ежемесячной
