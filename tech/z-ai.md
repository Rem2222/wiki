# Z.AI

**Сайт:** http://z.ai/

AI-ориентированная поисковая система и API-платформа, оптимизированная для LLM и AI-агентов. Предоставляет MCP-сервер для веб-поиска.

## Фишка

Search MCP из коробки для Claude Code на тарифах Z.AI — не нужны отдельные API-ключи Google/Bing.

## Возможности

- **MCP server:** `uvx @z-ai/mcp` — ставится одной командой
- **Native Claude Code integration:** поисковый инструмент доступен модели как нативный тул
- **Structured results:** с цитатами и источниками
- **Real-time search:** информация за пределами тренировочных данных модели

## Установка

```bash
uvx @z-ai/mcp
```

Затем добавить в конфиг MCP с API-ключом.

## Зачем

В статье упоминается, что Z.AI предоставляет поисковый MCP на своих тарифах, но лимиты «отнюдь не щедрые». Альтернатива — cc-websearch с собственными API-ключами.

## Нужно ли

✅ **Да, если** хочешь поиск в Claude Code без возни с API-ключами. Free-тарифа хватит попробовать.
❌ **Нет, если** предпочитаешь cc-websearch с собственными ключами (Google/Bing) — дешевле на больших объёмах.
