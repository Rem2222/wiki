# CloakBrowser

**CloakHQ/CloakBrowser** — Stealth Chromium с изменёнными fingerprint-ами на уровне C++ исходников.

- 49 source-level C++ патчей (canvas, WebGL, audio, WebRTC, GPU, CDP)
- `humanize=True` — мышиные кривые, тайминг клавиатуры, скролл
- 0.9 reCAPTCHA v3 score
- Проходит Cloudflare Turnstile, FingerprintJS, BrowserScan (30/30)
- Drop-in замена Playwright/Puppeteer
- `pip install cloakbrowser` / `npm install cloakbrowser`
- 14.1k ★, open source

**Для кого:** веб-скрейпинг, автоматизация сайтов с бот-детекцией.
**Роману:** нерелевантно, если только не понадобится обходить блокировки при парсинге.
