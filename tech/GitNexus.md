# GitNexus

**Тег:** #инструмент #кодинг #model辅助

## Что это
Веб-интерфейс для хостинга Git-репозиториев — как GitHub, только self-hosted. Можно поставить на свой сервер, бесплатно и под полным контролем.

## Ссылки
- Репозиторий: https://github.com/abhigyanpatwari/GitNexus
- Демо: https://gitnexus.vercel.app

## Фичи
- 🗂️ Хранить код — загрузка проектов, просмотр в браузере
- 🔀 Pull Request'ы — предлагать и мержить изменения
- 🐛 Issues — трекер багов и задач
- ⚙️ CI/CD — автоматические проверки при коммитах

## Технологии
Java (Spring Boot) + React (фронтенд)

## Лицензия
MIT

## Плюсы
- Приватный, бесплатный, полный контроль

## Минусы
- Менее развитое комьюнити чем Gitea/Forgejo
- Скромнее GitHub

## Альтернативы
[[tech/gitea]], [[tech/forgejo]], GitLab (self-hosted)

## Заметки
- Используется как **инструмент для кодинга и помощи модели в знании кода**
