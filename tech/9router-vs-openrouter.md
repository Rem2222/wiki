# 9Router vs OpenRouter

## 9Router

- **Тип**: Локальный open-source прокси (устанавливается на машину)
- **Репо**: github.com/decolua/9router
- **Сайт**: 9router.com
- **NPM**: npmjs.com/package/9router
- **Провайдеров**: 50+
- **Фокус**: Бесплатные tier-ы, $0 стоимость
- **Фичи**: Smart 3-tier fallback, account pooling, единый пул бесплатных API

**Главный посыл**: "никаких затрат — используешь то, что уже есть бесплатно"

## OpenRouter

- **Тип**: Облачный коммерческий сервис
- **Сайт**: openrouter.ai
- **Моделей**: Сотни
- **Фокус**: Платные модели, удобство, единый ключ
- **Фичи**: Model ranking, comparison, аналитика

**Главный посыл**: удобство единого ключа и выбор лучших моделей

## Сравнение

| | 9Router | OpenRouter |
|---|---|---|
| Расположение | Локальный | Облако |
| Деньги | $0 (free tiers) | Платный |
| Фокус | Лимиты free-tier аккаунтов | Платные модели |
| Тип | Open source | Коммерческий |

## Когда что использовать

- **9Router**: хочешь максимум из бесплатного, не хочешь платить
- **OpenRouter**: готов платить за удобство и лучшие модели
