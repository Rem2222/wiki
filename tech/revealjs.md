# reveal.js

**GitHub:** https://github.com/hakimel/reveal.js
**Сайт:** https://revealjs.com/

**Лицензия:** MIT
**Stars:** 71.5k ⭐
**Релиз:** v6.0.1 (апрель 2026)
**Стек:** Vite 8 + TypeScript 6

HTML-фреймворк для создания презентаций. Всё что нужно — браузер. Презентация пишется на HTML/CSS/JS/Markdown.

## Фишка

Полноценные презентации из Markdown одной командой. Git-friendly, CI-ready, бесплатно.

## Возможности

- **Nested slides** — вложенные слайды с вертикальной навигацией
- **Markdown support** — слайды в чистом Markdown
- **Auto-Animate** — плавные анимации между слайдами (как Powerpoint Morph)
- **PDF export** — экспорт в PDF одной кнопкой
- **Speaker notes** — заметки для докладчика
- **LaTeX (MathJax)** — математические формулы
- **Syntax highlighting** — подсветка кода
- **Extensive API** — плагины и кастомные расширения
- **React package** — `npm install reveal.js`

## Экосистема

- **Slides.com** — визуальный редактор от тех же авторов
- **Плагины:** highlight, math, search, notes, zoom
- **React-компонент** для интеграции в React-приложения

## Установка

```bash
npm install reveal.js
```

Или CDN:

```html
<link rel="stylesheet" href="https://unpkg.com/reveal.js@4/dist/reveal.css">
<script src="https://unpkg.com/reveal.js@4/dist/reveal.js"></script>
```

## Нужно ли

Полезно, если надо делать презентации для докладов/конференций — быстрее и удобнее чем Powerpoint/Keynote, особенно если работаешь в IDE.
