# playsvideo

**Тип:** Расширение Chrome для локального воспроизведения видео

**Ссылка:** https://chromewebstore.google.com/detail/playsvideo/leanfmncjikipbkbjghejpaenkpanimp

**Версия:** 0.1.0 (март 2026)

---

## Описание

Локальный видеоплеер в Chrome — воспроизводит любые видеофайлы прямо в браузере **без загрузки на сервер**.

Если Chrome не может воспроизвести формат нативно — перекодирует на лету через WebAssembly + Web Workers. Аудиокодеки автоматически транскодируются в AAC.

---

## Возможности

- Drag & drop или файловый диалог
- Встроенное извлечение субтитров
- Загрузка внешних субтитров (.srt, .vtt)
- Тёмная тема
- Open source: https://github.com/kzahel/playsvideo

---

## Установка

1. Перейти по ссылке в Chrome Web Store
2. Нажать **"Add to Chrome"**
3. Подтвердить установку
4. Иконка появится рядом с адресной строкой

---

## Когда использовать

- Нужно быстро посмотреть видео, не открывая отдельный плеер
- Файл в нестандартном формате, который не воспроизводится нативно
- Полезно для визуального анализа видео через Vision-модели
