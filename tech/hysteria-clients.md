# Hysteria 2 — клиенты для Windows и Android

Официальный список 3rd-party приложений с поддержкой Hysteria 2.

## Windows

| Клиент | Тип | Особенности |
|--------|-----|-------------|
| **Hiddify Next** | GUI | Мультиплатформенный авто-прокси. Простой импорт ссылок, поддержка правил. Бесплатно. |
| **Clash.Meta** | GUI | Правила, туннель на Go. Продвинутая маршрутизация. |
| **sing-box** | CLI/GUI | Универсальная прокси-платформа. Есть графические обёртки. |
| **NekoRay** | GUI | Легковесный. Поддерживает Hysteria 2, VLESS, Shadowsocks и другие. |
| **Furious** | GUI | PySide6, кроссплатформенный. Простой. |
| **v2rayN** | GUI | Классика. Поддерживает Hysteria 2 через core. |

### Рекомендация для Windows: **Hiddify Next**

Учитывая твой стек — Hiddify Next оптимален:
- Простая настройка (импорт ссылки)
- Системный прокси одной кнопкой
- Туннелирование по правилам
- Автообновление подписок
- Бесплатно, open source

**Альтернатива:** NekoRay — если нужно больше гибкости и ручной конфигурации.

## Android

| Клиент | Тип | Особенности |
|--------|-----|-------------|
| **Hiddify Next** | GUI | Тот же клиент, что на Windows. Авто-прокси, VPN-режим. |
| **NekoBox for Android** | GUI | Преемник Matsuri. Поддерживает Hysteria 2, VLESS, Shadowsocks, полный контроль. |
| **sing-box** | CLI/GUI | Есть Android-сборка. Лёгкая, мощная. |
| **Clash.Meta (Android)** | GUI | Clash-мета под Android (Clash Meta for Android). |
| **V2Box** | GUI | Мультипротокольный клиент. Простой интерфейс. |
| **Hy2NG** | GUI | Специализированный Hysteria2-клиент. Есть встроенный мастер сервера. |

### Рекомендация для Android: **Hiddify Next** (универсальность) или **NekoBox** (гибкость)

- **Hiddify Next** — один клиент на всё, синхронизация конфигов
- **NekoBox** — максимум контроля, если любишь копаться в настройках
