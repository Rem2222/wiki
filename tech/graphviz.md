# graphviz

Graphviz — open-source библиотека для визуализации графов на языке DOT.

Используется для:
- Диаграмм архитектуры
- Графов зависимостей
- Визуализации данных

Формат: `.gv` / `.dot` → PNG/SVG/PDF
