# Agentation

Enterprise-платформа для разработки, развёртывания и мониторинга AI-агентов. Позиционируется как **«операционная система для ИИ-агентов»**.

## Возможности

- **Визуальный конструктор (No-Code/Low-Code)** — drag-and-drop построение мультиагентных воркфлоу
- **Мульти-LLM роутинг** — GPT-4, Claude, Gemini, Llama, Mistral; разные модели для разных агентов
- **RAG и базы знаний** — PDF, Confluence, Notion, веб; Pinecone, Weaviate, Qdrant, pgvector
- **Agent Hub** — шаблоны: Customer Support, Sales Qualification, Data Analyst, IT Ticketing, HR Onboarding
- **Интеграции** — Salesforce, HubSpot, Zendesk, Intercom, Slack, Teams, PostgreSQL, Snowflake, BigQuery
- **Human-in-the-Loop** — остановка для подтверждения, полный аудит-трейл
- **Observability** — стоимость вызовов LLM, токены, latency по каждому агенту
- **Безопасность** — SOC 2 Type II, RBAC, E2E encryption, on-premise deployment

## Целевая аудитория

Enterprise (500+ сотрудников), которым нужна автоматизация процессов с корпоративной безопасностью.

## Конкуренты

Отличается от open-source (CrewAI, AutoGen, LangGraph) enterprise-безопасностью, on-premise, compliance.

## Прайсинг

Не публичный («Contact Sales»), ориентировочно от $1,000–5,000/мес.
