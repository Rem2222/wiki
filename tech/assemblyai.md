# AssemblyAI

API-first компания, предоставляющая модели и API для распознавания речи и понимания аудио.

**Основание:** 2017, Dylan Fox  
**Штаб-квартира:** Сан-Франциско  
**Инвесторы:** a16z, Insight Partners, Y Combinator  
**Лицензия:** Proprietary (API)

## Продукты

### Speech-to-Text API
- **Async Transcription** — высокоточная асинхронная обработка длинных аудио
- **Real-time Transcription** — потоковая через WebSockets
- Фичи: Speaker Diarization, Sentiment Analysis, Content Moderation, Topic Detection, Entity Detection, PII Redaction, Auto Chapters & Highlights

### LeMUR
(Leveraging Large Models to Understand Raw Speech) — фреймворк для применения LLM к распознанной речи. Позволяет задавать вопросы голосовым данным на естественном языке.

### Audio Intelligence
Набор специализированных моделей для глубокого анализа аудио (модерация, классификация, безопасность).

## Технологии

- **Архитектура:** Conformer (CNN + Transformer), end-to-end deep learning
- **Модель Universal-1:** обучена на 12.5M+ часов аудио, 99+ языков
- **SOC 2 Type II** сертификация

## API и SDK

Официальные SDK: Python, JavaScript/Node.js, Go, Ruby, .NET, Java

## Конкуренты

Google Cloud STT, Amazon Transcribe, Azure Speech, Deepgram, Rev.ai, OpenAI Whisper
