# Caveman

**GitHub:** https://github.com/JuliusBrussee/caveman

Skill для Claude Code, который заставляет модель отвечать коротко и по делу, отбрасывая вежливости, рассуждения и мета-комментарии. Сокращает выходные токены на ~75%.

## Фишка

Экономит ~75% токенов на выходе. Говорит как пещерный человек, но по существу.

## Как работает

Форсирует системный промпт, который запрещает модели:
- Вежливости и извинения
- Развёрнутые объяснения без необходимости
- Мета-комментарии
- Повторения

Вместо этого — короткие прямые ответы.

## Эффект

- **~75% меньше выходных токенов** → в 4 раза дешевле
- **Быстрее генерация** — меньше времени ждать ответ
- **Содержание не страдает** — факты и логика те же, просто без воды

## Установка

```bash
git clone https://github.com/JuliusBrussee/caveman
# или через реестр плагинов
claude plugin marketplace add JuliusBrussee/caveman
claude plugin install caveman@caveman
```

## Зачем

Выходные токены — дорогие. Caveman их радикально сокращает. Даже если не жалко денег на подписку — экономится время генерации.

## Нужно ли

✅ **Да, если** используешь pay-per-token модели или дорожишь временем.
❌ **Нет, если** тебе нужны развёрнутые рассуждения модели (contradicts SDD/Spec-driven approaches).
⚠️ Per-project или включать/выключать по сессии.
