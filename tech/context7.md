# Context7

**Сайт:** https://context7.com/

MCP-сервер, предоставляющий AI-агентам доступ к актуальной документации библиотек и фреймворков. Решает проблему галлюцинаций API — модель получает реальную документацию вместо опоры на тренировочные данные.

## Фишка

Единственный тул, который даёт модели актуальную документацию по любой библиотеке на любом языке через MCP. Без него модель врёт про API.

## Возможности

- **Search documentation:** модель ищет документацию по запросу и получает структурированный ответ с цитатами
- **200+ библиотек:** React, Stripe, FastAPI, LangChain, Python stdlib, Go stdlib и т.д.
- **Always up-to-date:** документация тянется в реальном времени, не из тренировочных данных
- **Claude Code integration:** официальный плагин в реестре плагинов Claude Code

## Цены

- **Free:** до ~500-1000 запросов/мес (для индивидуального использования)
- **Pro:** $20-30/мес — больше запросов, приоритет
- **Team/Enterprise:** кастом

## Установка

Через официальный реестр Claude Code. Также можно добавить вручную в `.mcp.json`.

```bash
claude plugin install @context7/mcp
```

## Зачем

Без Context7 модель будет выдумывать API библиотек. С ним — работает с реальной документацией. Категорически рекомендуется для любого программистского проекта.

## Нужно ли

✅ **Однозначно да.** Ставить глобально — будет полезно в каждом проекте.
