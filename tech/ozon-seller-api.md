# Ozon Seller API — инструменты

Подборка Open Source инструментов для работы с Ozon Seller API (для продавцов).

## MCP-серверы (для AI-агентов)

- **[ozon-mcp](https://github.com/PCDCK/ozon-mcp)** ★6 — MCP-сервер для Ozon Seller API + Performance API. Подключает AI-агента к кабинету продавца.
- **[mcp-server-ozon-seller](https://github.com/dontsovcmc/mcp-server-ozon-seller)** ★2 — альтернативный MCP-сервер для Ozon Seller API.

## Python

- **[OzonAPI](https://github.com/a-ulianov/OzonAPI)** ★22 — асинхронная библиотека с лимитами, автоповторами запросов, документацией и примерами.
- **[python-ozon-api](https://github.com/mephistofox/python-ozon-api)** ★21 — асинхронная Python библиотека.
- **[OzonSellerApi](https://github.com/KlimAlexandrovich/OzonSellerApi)** ★15 — модуль для получения данных продаж.
- **[ozon-seller](https://github.com/irenicaa/ozon-seller)** ★2 — клиент для Ozon Seller API.
- **[marketplace_apis](https://github.com/packify-org/marketplace_apis)** ★9 — async обёртка для российских маркетплейсов (Ozon, WB, Yandex).
- **PyPI**: `ozon-api` v1.6.1, `ozon-seller` v1.0.0

## Go

- **[ozon-api-client](https://github.com/diPhantxm/ozon-api-client)** ★48 — Golang клиент Seller API.
- **[ozon-api](https://github.com/ghost-gopher/ozon-api)** ★2 — Go клиент Common API + Performance API.

## PHP

- **[ozon-seller](https://github.com/gam6itko/ozon-seller)** ★115 — самый популярный PHP клиент для Seller API (51 fork).
- **[wb-ozon-api](https://github.com/biohazardhome/wb-ozon-api)** ★9 — Wildberries + Ozon Seller API для Laravel.
- **[ozon-marketplace](https://github.com/gam6itko/ozon-marketplace)** ★5 — ещё одна PHP библиотека.

## TypeScript / JavaScript

- **[ozon-daytona-seller-api](https://github.com/salacoste/ozon-daytona-seller-api)** ★9 — полный TypeScript SDK, 278 методов, 33 категории.
- **[OzonFromGAS](https://github.com/googlesheets-ru/OzonFromGAS)** ★33 — Ozon Seller API для Google Apps Script.
- **[ozon-api-client-for-gas](https://github.com/mityayka1/ozon-api-client-for-gas)** ★5 — ещё один клиент для GAS.

## C#

- **[OzonSellerApi](https://github.com/feeleen/OzonSellerApi)** ★9 — клиент на C# для Seller API.
- **[OzonSellerApi](https://github.com/koshaliev/OzonSellerApi)** ★6 — ещё одна C# библиотека.

## Документация

- **[ozon-seller-api-docs](https://github.com/DragonSigh/ozon-seller-api-docs)** ★6 — 66 файлов документации по Ozon Seller API в Markdown (товары, заказы, аналитика).
- **[ozon-performance-api-docs](https://github.com/DragonSigh/ozon-performance-api-docs)** — документация рекламного API в Markdown (11 файлов).

## Парсеры и инструменты

- **[ozon-parser](https://github.com/NurjahonErgashevMe/ozon-parser)** ★79 — парсер товаров с Ozon.ru, многопоточность, экспорт в Excel/JSON, Telegram бот.
- **[ozon-pinneaples](https://github.com/Churkashh/ozon-pinneaples)** ★57 — сбор ананасов ozon.ru (парсер акций/скидок).
- **[mpsEx](https://github.com/nlkli/mpsEx)** ★7 — Chrome расширение для парсинга каталогов Ozon и Wildberries.
- **[mpp-core](https://github.com/f1lcry/mpp-core)** ★3 — модульный движок автоматизации маркетплейсов (парсинг 1688, AI-описания, выгрузка через Ozon API).
- **[ECommerce](https://github.com/Lev4and/ECommerce)** ★5 — ASP.NET Core + Vue.js решение.
